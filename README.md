🧮 Multi-Purpose Calculator Web App

A simple but powerful web-based calculator built with PHP, HTML, CSS, and JavaScript.
This project demonstrates basic mathematical problem-solving using server-side PHP logic.

🚀 Features

✔ Quadratic Equation Solver

✔ Simple Calculator (Add, Subtract, Multiply, Divide)

✔ BMI Calculator

✔ Compound Interest Calculator

✔ Linear Equation Solver

✔ Simultaneous Equation Solver

✔ Simple Interest Calculator

✔ Input validation and error handling

✔ Clean UI with responsive design

✔ Beginner-friendly PHP backend logic

🛠️ Technologies Used
PHP (Core Logic)

HTML5 (Structure)

CSS3 (Styling)

JavaScript (Optional UI enhancements)

XAMPP (Local Server Environment)

📐 Mathematical Functions Implemented

Quadratic Formula Solver

Basic Arithmetic Operations

Linear & Simultaneous Equations

Financial Calculations (Simple & Compound Interest)

BMI Calculation System

Input Validation System

⚙️ How to Run the Project Locally

Install XAMPP

Move project folder to:

C:\xampp\htdocs\Calculator

Start Apache in XAMPP Control Panel

Open browser and visit:

http://localhost/Calculator
